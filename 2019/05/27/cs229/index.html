Problem Set #1 监督学习(Supervised Learning)

1. Logistic regression

(a) 已知逻辑回归的cost function如下：

并且$y^{(i)} \in \{-1, + 1\}, h_{\theta}(x) = g(\theta^Tx), g(z) = 1/(1+e^{-z}).$证明对于此cost function的Hessian $H$，对任意向量$z$, 有：

解：$g(z)一阶导数：g’ = g(z)(1 - g(z))$

先求$J(\theta)$一阶导数，

$Hessian H_{ij} = {\partial J(\theta)}/{\partial \theta_i} \partial \theta_j$

So:

$因为g(z)\in{(0,1)}, 所以 g(z)(1 - g(z)) \ge 0, \sum^{n}_{i=1}\sum^{n}_{j=1}z_ix^{(k)}_{i}x^{(k)}_{j}z_j = (\sum^{n}_{i=1}z_ix^{(k)}_{i}) (\sum^{n}_{j=1}z_jx^{(k)}_{j})= (z^Tx)^2 \ge 0$

$所以 z^THz \ge 0$

(b)逻辑回归的牛顿法求解：

两数据文件包括输入$(x^{(i)} \in R^2)$ 以及label $(y^(i)) \in \{-1, +1\}$，实现一个牛顿方法来优化$J( \theta)$,以及来拟合一个逻辑回归模型。初始化牛顿方法参数$\theta = \vec0$,求最后拟合的系数$ \theta$的值大小？

牛顿方法迭代公式: $ \theta := \theta + H^{-1} \nabla_\theta J(\theta)$

并且$H^{-1} = \frac{1}{m}\sum^{m}_{k=1}g(z)(1 - g(z)) x^{(k)}_{i}x^{(k)}_{j}, {\partial J(\theta)}/{\partial \theta_i} =-\frac{1}{m}\sum^{m}_{k=1}(1 - g(z)) y^{(k)}x^{(k)}_{i}$
