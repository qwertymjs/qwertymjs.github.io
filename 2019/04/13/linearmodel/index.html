1.6 Instantiate Loss Function

  • MSELoss: Mean Squared Erros

1.7 Optimizer Class

  • $\theta = \theta - \eta \nabla_{\theta}$

  • $\eta:$ learning rate
  • $\nabla_{\theta}:$ parameters’s gradients
