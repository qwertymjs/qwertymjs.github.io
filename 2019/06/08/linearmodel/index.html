线性回归就是用一组输入变量的线性组合，来建立与输出的关系。即期望用一条最佳的直线（被称为回归线）来表示因变量($Y$)和一个或多个自变量($X$)之间的关系。图例:带一个自变量的线性回归

其中，$\begin{pmatrix} x_1& x_2 & \dots & x_p \end{pmatrix}$表示自变量(默认$x_0$ 值为1，实际上$x$在计算时是p+1的向量)；$y$是因变量；$w$为参数向量；$w_i$表示对应自变量的权重，$w_0$是偏置项(又称为截距)。

关于参数$w$：

  1. 在物理上可以这样解释：在输入特征向量之间相互独立的前提下，$w_i$反映自变量$x_i$对因变量影响权重，$w_i$越大，说明$x_i$对结果$y$的影响越大。
  2. 通过特征前面的参数$w_i$，可以很直观的看出哪些特征分量对结果的影响比较大。

1.4 概率角度：最小二乘法

一般地，机器学习中不同的模型会有相应的目标函数。而回归模型（尤其是线性回归类）的目标函数通常用平方损失函数作为优化的目标函数（即真实值与预测值之差的平方和）。为什么要选用误差平方和作为目标函数呢？

样本数据与真实数据之前肯定存在一定的噪声(可能是测量方法或者记录导致的)，噪声都是无法避免的。假设给定一个输入样本$x^{(i)}$，得到预测值$w^T x^{(i)}$与真实值$y^{(i)}$之间存在误差，记为$\epsilon^{(i)}$。那么，目标值$\hat{y}$表示如下:

而这里假设误差$ \epsilon^{(i)}$服从标准高斯分布是合理的。解释如下：

  • 回归模型的最终目标是建立自变量$x$与结果$y$之间的关系（通过函数表达式），希望通过$x$能较准确的表示输出$y$。

  • 而在实际应用场景中，很难甚至不可能把导致$y$结果的所有变量（特征）都找出来，并放到回归模型中。那么模型中存在的$x$通常认为是影响$y$最主要的变量集合（又称因子, 在ML中叫做特征集）。根据中心极限定理，把那些对结果影响比较小的变量（假设独立同分布）之和认为服从正态分布是合理的。

所以我们假设数据是带有服从高斯分布的噪声，$\epsilon \sim \mathcal{N}(0, \sigma^2)$，

因此：$y|x$也是服从高斯分布的，并且$y \sim \mathcal{N}(w^Tx, \sigma^2)$

那么$x$和$y$的条件概率可表示为：

根据$\mathcal{p}(y^{(i)}|x^{(i)},w) $的单条样本预测概率，模型的最终目标是希望在全部样本上预测最准，也就是概率积最大，这个概率积就是似然函数。优化的目标的函数即为最大化似然函数(极大似然估计)，由于样本数据是独立同分布的，所以：

将$y|x$的高斯概率分布函数代入$L(w)$:

由于$\hat{w} = \operatorname{argmax}L(w) = \operatorname{argmin}\sum_{i=1}^N(y^{(i)}-w^Tx^{(i)})^2$

结论：从概率角度可得：最小二乘法等价于数据噪声为高斯分布的MLE

1.5 回归模型与正则化

线性回归参数$w$的解析解：

看起来似乎很简单，但是在实际使用的过程中会有不少问题。

  • 从数学角度看，$X^TX$可能不可逆，导致目标函数最小化导数为零时方程有无穷解，没办法求出最优解。尤其在$p > N$时，必然存在这样的问题。
  • 从模型看那就是过拟合了。
  • 过拟合有三种方式缓解：一个是增加数据量，一个是降维(特征提取，降低特征维度)，最后一个就是正则化了。
  • 所谓正则化就是对参数$w$做一些限制，使得它的最优解空间变小。

不过，特别是在像生物或者医学等通常需要和人交互的领域，稀疏的解除了计算量上的好处之外，更重要的是更具有“可解释性”。比如说，一个病如果依赖于 5 个变量的话，将会更易于医生理解、描述和总结规律，但是如果依赖于 5000 个变量的话，基本上就超出人肉可处理的范围了。

稀疏约束最直观的形式应该是约束0范式，如上面的范式介绍，$w$的0范式是求$w$中非零元素的个数。如果约束$\lVert w \Vert_0 \le k$，就是约束非零元素个数不大于k。不过很明显，0范式是不连续的且非凸的，如果在线性回归中加上0范式的约束，就变成了一个组合优化问题：挑出$\le k$个系数然后做回归，找到目标函数的最小值对应的系数组合，是一个NP问题。

相比之下，l2-ball 就没有这样的性质，因为没有角，所以第一次相交的地方出现在具有稀疏性的位置的概率就变得非常小了。这就从直观上来解释了为什么l1regularization 能产生稀疏性，而 l2regularization 不行的原因了。

2. 逻辑回归 logistic regression

分类问题，而且是二分类，是机器学习中一个基本的学习场景。例如判断邮件是不是垃圾邮件，用户对商品是不是该买等等。数学定义上就是：给定$m$个数据样本$(x^{(1)}, y^{(1)}),(x^{(2)}, y^{(2)}),\dots,(x^{(m)}, y^{(m)})$,其中$x^{(i)} \in R^{n}$, 表示第$i$个样本的n维特征向量，$y^{(i)} \in \{0, 1\}$表示样本的目标值是正样本(1)还是负样本(0)。自然而然假设目标值是服从伯努利分布，有$y|x;\theta \sim Bernoulli(\phi)$。逻辑回归模型通过一个sigmoid函数$\sigma(x) = 1/ (1 + e^{-x})$表示输入$x^{(i)}$与目标值为正样本的概率。

直观上来看，参数$\theta$对$x^{(i)} $的$n$个维度做不同的加权，从而算出值$\theta^Tx^{(i)}$,这个值通过sigmoid函数压缩到$\{0,1\}$之间，作为$x^{(i)}$对应的目标值为正样本的概率。参数$\theta$是通过极大似然估计(Maximum Likelihood Estimation,简称MLE)来找到最佳的值。

多说两句，MLE是属于频率派的做法。在频率派中，认为求解的参数$\theta$是未知的常量，样本数据$x$是随机变量，并且是服从概率$p(x|\theta)$独立同分布(iid), $\theta$的值是使似然函数$p(X|\theta)$达到最大的$\theta$值，也就是选择使观察数据集出现概率最大的$\theta$的值。另外似然函数的负对数也叫损失函数。由于负对数是单调递减的函数，所以最大化似然函数等价于最小化损失函数，所以频率派最后都转化成一个优化问题。
最大化似然函数：$\max_{\theta}\sum_{i=1}^m \log p(y^{(i)}|x^{(i)},\theta)$:
当$y \in \{0, 1\}$时：$\max_{\theta}\sum_{i=1}^m \log p(y^{(i)}|x^{(i)},\theta) = \sum_{i=1}^m (y^{(i)}\log (\sigma(\theta^Tx^{(i)})) + (1 - y^{(i)}) \log (1 - \sigma(\theta^Tx^{(i)}))) $
所以极大化转化成极小化损失函数：$J(\theta) = -\sum_{i=1}^m (y^{(i)}\log (\sigma(\theta^Tx^{(i)})) + (1 - y^{(i)}) \log (1 - \sigma(\theta^Tx^{(i)}))) $
PS:当$y \in \{-1, +1\}$时：$\max_{\theta}\sum_{i=1}^m \log p(y^{(i)}|x^{(i)},\theta) = -\sum_{i=1}^m \log (1 + e^{y^{(i)}\theta^Tx^{(i)}})$
相对的：$J(\theta) = \sum_{i=1}^m \log (1 + e^{y^{(i)}\theta^Tx^{(i)}})$

2.1 $L1 vs L2$正则化

求解上面的损失函数之前，我们先了解下正则化。正则化的作用是用来避免模型过度拟合训练数据(overfitting)。例如假设仅仅时最小化$J(\theta)$,而对$\theta$没有任何限制，优化方法可以随意调整$\theta$各个维度的值使得模型最好的符合m个样本点，从而最小化$J(\theta)$。这样得到的$J(\theta)$各个维度的绝对值通常很大：一些大正数，一些大负数。这种模型虽然在训练数据有很好的表现，但是对新的样本预测时，这些绝对值很大的取值就会使得预测值偏离真正值很远。

正则化定义

通常做法，我们会在$J(\theta)$加上一个关于$w$大小的项，称作正则化因子。一般选用$\theta$的模。

其中$\lVert\theta\rVert$是L1模，称为L1正则化

或者时L2模，称为L2正则化

无论是L1,L2还是其他模做正则化，基本原理都是一样，即在最小化损失函数$J(\theta)$的同时，还有考虑$\theta$的模带来的贡献，避免$\theta$取一些绝对值很大的值。

L1与L2正则化区别

一句话概括就是L1正则化通常导致稀疏的模型，也就是使得$\theta$中有更多维度为0，而这些为0的维度就表示不是很相关的特征，从而起到了特征选择(Feature Selection)的作用。
